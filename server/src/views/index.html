üì° Getting Started

This API provides endpoints for managing mathematical research, documents, events, and team collaboration. Public endpoints (marked with PUBLIC) are accessible without authentication. Protected endpoints require a JWT token.

To get a token, login via POST /auth/login with admin credentials.

üåç Public Endpoints

News Public

GET /news Get all news articles
GET /news/:id Get news by ID

Events Public

GET /events Get all events
GET /events/upcoming Get upcoming events
GET /events/:id Get event by ID

Authentication Public

POST /auth/login Admin login (get JWT token)
POST /auth/register Register new admin

üîí Protected Endpoints (Require Authentication)

Dashboard Protected

GET /admin/dashboard/stats Dashboard statistics
GET /admin/dashboard/user-data Current user data
GET /admin/dashboard/categories Time tracking categories
GET /admin/dashboard/notifications User notifications

Documents Protected

GET /documents Get all documents
GET /documents/stats Document statistics
POST /documents Create document
PUT /documents/:id Update document
DELETE /documents/:id Delete document

Researchers Protected

GET /researchers Get all researchers
GET /researchers/leaderboard Top contributors
POST /researchers Create researcher

Repositories Protected

GET /repositories Get all repositories
GET /repositories/user User's repositories
POST /repositories Create repository

üöÄ Quick Start

1. Test Public Endpoints

Try fetching news: GET http://localhost:3000/news

2. Login to Access Protected Endpoints

Send POST request to /auth/login with your admin credentials

3. Use JWT Token

Add token to requests: Authorization: Bearer YOUR_TOKEN
